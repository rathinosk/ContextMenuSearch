Add Search Engines Manually

To add a new search engine, replace the search term in the search engine's URL with "TESTSEARCH" and copy the entire URL (including http:// or https://) into the 'Link' field below.
For example, to add Bing, search for any term in Bing, replace the search term in the URL with "TESTSEARCH", and paste the modified URL here.


💡 Tip 1: You can use "%s" to represent search terms. This is useful when copying from Chrome settings.
Tip 2: To open multiple links with a single option, separate the links with a space.
For example, to open Google and Bing search results together, use:
"http://www.google.com/search?q=TESTSEARCH http://www.bing.com/search?q=TESTSEARCH"
Tip 3: If you don't want the selected term to be URL encoded, use "NOENCODESEARCH" instead of "TESTSEARCH".

About Context Menu Search

Developer: Rathinosk | Extension home


Context Menu Search is a simple extension that allows you to search selected text with a search engine of your choice.
This extension does not communicate with any websites or services.
