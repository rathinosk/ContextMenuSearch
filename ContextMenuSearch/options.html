Add Search Engines Manually

To add a new search engine, replace the search term in the search engine's URL with "TESTSEARCH" and copy the entire URL (including http:// or https://) into the 'Link' field below.
For example, to add Bing, search for any term in Bing, replace the search term in the URL with "TESTSEARCH", and paste the modified URL here.


💡 Tip 1: You can use "%s" to represent search terms. This is useful when copying from Chrome settings.
Tip 2: To open multiple links with a single option, separate the links with a space.
For example, to open Google and Bing search results together, use:
"http://www.google.com/search?q=TESTSEARCH http://www.bing.com/search?q=TESTSEARCH"
Tip 3: If you don't want the selected term to be URL encoded, use "NOENCODESEARCH" instead of "TESTSEARCH".

About Context Menu Search

Version 1.0.1

Developer: Rathinosk | Chrome Web Store Page


Context Menu Search is a simple extension that allows you to search selected text with a search engine of your choice.
This extension does not communicate with any websites or services -- only the search engine you choose is contacted.

Change-log

1.0.1: Minor bug-fixes and improvements.

  • Improved error handling and increased console logging.
  • Added a new option to reset all options to the default values.


1.0.0: This is the first public release -- a major update with numerous bug fixes, new features and improvements.

  • Revamped tab handler -- fixed bug where search did not work from within a PDF viewer tab.
  • Fixed multiple scenarios that caused the extension to not load the configuration. Reloading will generally fix most configuration loading problems.
  • Fixed the option to show "Options..." at the bottom of the context menu.


pre-1.0.0: Internal working versions with refactoring and bug-fixes to the original (v2.99) Context Menu Search extension.
These versions were not released to the Chrome Web Store.
